The Computer and Projector

Getting Started​

Turn on the Computer with the button on the front.

The desktop should show up on the monitor on the desk. If it does not make sure it is switched on and has a Blue/Orange light showing on the right corner

The Extron box to the right of the monitor will display numbers on the screen if the computer is outputting signal to the monitor and projector. If there is not a video signal it will read "No Signal"

Turning The Projejector on​

Press the ON Button on the remote to turn the projector on. It will take a minute to warm up the lamp and display a picture.

The correct input should already be selected. If there is no picture after a couple minutes check that the input is set to "Normal" Video

Remote Inputs:

  • "Normal" Video
    • Video from the Grey Matrix selector
  • "Desktop" RGB2 - Unused
  • RGB1 - Unused

Changing Video Sources​

The Grey Matrix Switcher controls all inputs to the projector.

To change sources select the desired Input on the top row, then select the Projector Output on the bottom row and press Enter on the right side.

Matrix Inputs:

  • Sanctuary - Input 1
    • Video from the broadcast system in the Sanctuary
  • Computer - Input 2
    • Video from the fellowship hall computer. Duplicating the desktop for ease of use.
  • DVD - Input 3
    • Signal from the DVD player to the right
